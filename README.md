# music-sd
netease and qq music search and download  
当前支持网易云音乐和QQ音乐的搜索和下载

## install
1. 编译安装

   安装go的基础的环境
```sh
go get -d github.com/scroot/music-sd
cd $GOPATH/src/github.com/scroot/music-sd
go build .
```
2. 直接使用编译好二进制包
